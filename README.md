# Inventory Management System with AI Integration

## Features
- Product management
- Stock in/out tracking
- Inventory queries and reporting
- Generative AI support for natural language queries and suggestions

## Structure
- `server`: Node.js backend (Express, SQLite, AI integration)
- `client`: React frontend (UI for inventory management)

## Quick Start
1. Install dependencies in both `server` and `client`.
2. Start backend and frontend servers.
3. Access the web UI and try AI-powered queries.

## AI Integration
- Uses OpenAI API (or similar) for natural language support.
